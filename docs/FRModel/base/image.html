Module Info¶

class FRModel.base.image.Image(img: PIL.Image.Image)¶

Bases: object

This class will hold information about the image being used.

Use from_image to read an image.

This encapsulates the Pillow Image class to provide the required interfaces only.

class PartitionMethod(value)¶

Bases: enum.Enum

Different ways to treat edge cases.

PAD: Add black padding to the edges. CROP: Crop out edges, will cause inconsistent sizes REMOVE: Remove anything that crosses an edge.

partition(window: Tuple = (100, 100), stride: Tuple = (50, 50), edge_method: PartitionMethod = <PartitionMethod.PAD: (0, )>) → array[Image]¶

Partitions the image into fixed sizes, with definable stride

For example, if we have window = (100, 100), stride = (50, 50), the algorithm will move a 100x100px window along the image, extracting images

Parameters

  • edge_method – Defines the method to use to account for windows that exceed edges

  • window – A Tuple of (x-axis, y-axis) size in px

  • stride – A Tuple of (x-axis, y-axis) stride size in px
